Python模块安装Pygame安装

参考书籍《Python编程从入门到实践》

一、使用pip安装Python包

大多数较新的Python版本都自带pip，因此首先可检查系统是否已经安装了pip。在Python3中, pip有时被称为pip3,在Linux和OS X系统中检查是否安装了pip打开一个终端窗口，并执行如下命令：

Python2.7中：

$ pip --version

Python3中：

$ pip3 --version

如果不存在，则需要安装pip，在终端中安装pip3/pip的命令为

$ sudo apt install python3-pip
$ sudo apt install python-pip

二、在Linux系统中安装Pygame

如果使用Python3的版本，需要执行两个步骤：1）安装Pygame依赖的库，下载并安装Pygame。

执行下面的命令安装Pygame依赖的库（如果我们开始终端会话时使用的是命令Oython3.5，请将python3-dev替换为python3.5-dev）：我的Python3对应的版本为3.4.7

$ sudo apt-get install python3-dev mercurial

$ sudo apt-get install libsdl-image1.2-dev libsdl2-dev libsdl-ttf2.0-dev

如果我们需要启用Pygame的一些高级功能，如添加声音的功能，可安装下面这些额外的库：

$ sudo apt-get install libsdl-mixer1.2-dev libportmidi-dev
$ sudo apt-get install libswscale-dev libsmpeg-dev libavformat-dev libavcodec-dev
$ sudo apt-get install python-numpy

三、比较容易有坑的部分

按照书中的说法，尝试运行以下命令

$ pip install --user hg+http://bitbucket.org/pygame/pygame

或者

$ pip3 install --user hg+http://bitbucket.org/pygame/pygame

是有可能出错的，在通过搜索资料后，查询到博文地址

http://www.cnblogs.com/gengcx/p/6665800.html

采用如下的安装流程，接下来执行的命令如有必要，将pip替换成为pip3

$ hg clone https://bitbucket.org/pygame/pygame
$ cd pygame
$ python3 setup.py build
$ sudo python3 setup.py install

当然，如果按照书中的命令行提示能够执行成功是最好的。

完
